###### 5. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let count = 0;
const nums = [0, 1, 2, 3];

nums.forEach((num) => {
  if (num) count += 1;
});

console.log(count);
```

- A: 1
- B: 2
- C: 3
- D: 4

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C

<i> 
In the loop, for each num, it checks if (num). In JavaScript, 0 is considered a falsy value, and any non-zero number is considered truthy.
for the first element 0, the condition is not met (falsy), so count remains 0.
For the 2nd, 3rd, and 4th  elements 1, 2, and 3, the condition is met (truthy), so count is incremented by 1.
After iterating through all elements, the value of count is 3.

</i>

</p>
</details>
